# 상관 분석 (correlation analysis)
- 상관관계 두 연속형 변수(continuous variable)의 상관관계를 분석
- 다변량 통계분석의 경우 두 변수 간의 상관관계에 아주 많이 의존
  - 선형 상관 / 비 선형 상관
  - 데이터는 등간 또는 비율 척도
- 가설
  - 귀무가설 : 두 변수는 관계가 없다.
    - H0 : r = 0
  - 대립가설 : 두 변수는 관계가 있다.
    - Hl : r != 0
인과 관계(변인이 다른 변인의 원인)와 상관관계(변인과 변인의 변화 관계) 상관 관계를 인과 관계로 해석하면 안됨.

## 상관분석과 상관계수
- 선형성(Linearity)
- 등분산성(Homoscedasticity)
- 정규성(Normality)
- 독립성(Independence) : 다중공선성

## 상관분석 종합(correlation anaysis)
- 연속적 데이터에 적용
- 평가
- PCA
- 회귀분석의 정당성을 부여하고 자기상관과 다중공선성

## 상관분석의 종류
- Person 의 상관계수(correlation coefficient)
  - 두 변수간의 선형 종속성을 재는 측도로서 -1 ~ +1 사이에서 값을 가짐.
  - 절대값이 1에 가까울수록 두 변수 간의 선형 종속성이 강함을 의미.
  - 변수 간에 명백한 종속 관계가 있더라도 선형 종속성을 갖지 않으면 상관 계수 값은 0에 가까운 값이 됨.
  - 변수의 크기 간 연관성을 분석하면 선형 종속성을 넘어 단조적 관계성(monotonicity)에 대한 분석 가능

## 회귀분석
- 하나 이상의 독립변수들이 종속변수에 미치는 영향을 추정할 수 있는 통계 기법
- 회기방정식 -> 회귀선을 추정
- 통계적 추론 결과를 예측(prediction)에 이용하는 분석 반법 중 가장 기본이 되는 회귀분석, 로지스틱 회기문석
- 상관분석 결과에 의해 정당성을 확보할 수 있고, 해석기 용이하며 추론이 쉬움
- 회귀함수가 실제로 선형이 아닌 경우 예측력이 상당히 떨어질 수 밖에 없음
- 다항회귀, 비선형회귀 혹은 비모수적 함수추정 기법든
- 독립변수(설명변수)와 종속변수(반응변수)가 모두 연속형 변수(등간 척도 또는 비율 척도)

## 회귀분석의 종류
- 단순선형회귀
  - 회귀함수가 선형임을 가정, 독립변수가 하나인 경우
  - 회귀계수의 의미: x 의 값이 한 단위 증가함에 따라 y의 값에 생기는 평균 변화량(partial regression coefficients)
- 다항회귀
- 로지스틱 회귀
- 곡선회귀(포물선) : B-spline 회귀(능형회귀) : 과적합의 위험
- 분류나무, 회귀나무
  - 변곡점
  - 잔차의 제곱이 가장 작아지는 지점까지
